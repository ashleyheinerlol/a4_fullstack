What does the "Full" in Full Stack mean?
			Full means that not only do you develop the front end of the server,
			you also develop the back end. The front is what you see when you're 
			looking at the server. The back end is on the server computer and it 
			delivers content from the front end server.
What does the "Stack" in Full Stack mean?
			The "stack" in Full Stack is a stack of programs that are used.
			On the front end, which we learned about already, you will see 
			HTML 5, CSS3, and JavaScript. on the back end they use PHP, and MySQL.
What does "LAMP Stack" mean?
			LAMP is the 4 programs or "stack" for our web page server
			LAMP is an acronym for:
			L = Linux, which is like the instructions for our server opperating system
			A = Apache, or our server
			M = MySQL, the data base we use
			P = PHP, which is a programming language used on the back end
What is DB?
